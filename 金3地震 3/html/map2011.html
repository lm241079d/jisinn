災害の概要

2011年 東日本大震災

発生日時 3月11日 14時46分
死者数 / 行方不明者数 死者: 19,747人 / 不明: 2,556人 (国土交通省)
多かった死因 津波による溺死 (関連統計DB)
概要

東日本大震災では、地震そのものだけでなく、津波や原子力発電所事故によって、より多くの災害が引き起こされた。
福島第一原子力発電所の事故により、被災者は混乱をさらに深め、家を失い故郷を離れざるを得ない人も多くいた。
また、岩手県宮古市にあった世界最大級の防潮堤（高さ10m）をも越える津波が街を飲み込んだことで、「堤防やダムで災害を防ぐ」という考え方に大きな見直しが求められた。
以後、災害への対応は「防ぐ」から「被害を最小限に抑える」ことを重視する「減災」の考え方へと大きく転換した。
